# Resumesy — Privacy-First Free Resume Builder

Resumesy is a lightweight, browser-based resume builder focused on privacy and speed.  
All resume editing and PDF generation happen entirely in your browser — no data is uploaded or stored on a server.

## Why Resumesy
- **Local-first:** Resume data is created and processed on your device.
- **No uploads:** Content is never sent to external servers.
- **Live preview:** See changes instantly as you edit.
- **Print-ready PDFs:** Export professional, application-ready resumes locally.
- **Privacy-first:** No tracking, analytics, or background requests.

Resumesy is ideal for building modern resumes without sharing personal data with third-party platforms.

## Templates
Resumesy includes multiple professionally designed resume templates built for clarity and ATS compatibility. (20+ templaets available for free)

- Sidebar, compact, timeline, and creative layouts  
- Clean typography with balanced spacing  
- Sections render only when content exists (no empty blocks)  
- Designed for single-page and multi-page resumes  
- Optimized for print and PDF export  

Templates update instantly as you edit content and styling.

## Features
- Live resume preview while editing  
- Multiple professional templates (sidebar, compact, timeline, creative)  
- Section manager (add, hide, reorder sections)  
- Support for custom sections  
- Skill lists and categorized skills  
- Color and typography controls  
- Local PDF export (print-optimized)  
- Templates render only sections with content  

## How it works
- Built with **React**, **TypeScript**, and **Vite**
- Resume data lives only in browser memory during the session
- PDF export is generated client-side
- No use of server-side rendering or remote APIs

All functionality runs directly in the browser.

## Usage
1. Open the webapp in a modern browser.
2. Choose a template and start editing.
3. Customize sections, styling, and layout.
4. Export a print-ready PDF locally.


## Privacy model
- Resume content stays in browser memory only
- No localStorage, IndexedDB, or remote persistence
- No analytics, telemetry, or tracking scripts


## Sharing
Resumesy includes quick share buttons that prefill a message linking to the hosted demo — no resume data is shared.
